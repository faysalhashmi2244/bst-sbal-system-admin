Node-Based Affiliate System Project Summary

Project Overview

This project implements a blockchain-powered affiliate management system using smart contracts. The system enables transparent and automated referral reward distribution with advanced incentive mechanisms.

Key Features:

  • Seven-level MLM referral structure with configurable reward percentages
  • Direct referral rewards (10% by default)
  • Bulk referral bonus system (10% after every 10 referrals)
  • Prosperity Fund for community rewards (collects 10% of purchases)
  • Rewards-based discount system for node purchases
  • Admin panel for contract management

Smart Contract Implementation

NodeToken Contract

An ERC20 token used for purchasing node packages and distributing rewards.

NodePackages Contract

The main contract that handles:

  • Node package management (creation, updating, activation/deactivation)
  • User node purchases and reward calculations
  • Referral relationship tracking and reward distribution
  • Seven-level MLM structure management
  • Bulk referral bonus system
  • Prosperity Fund collection and distribution
  • Rewards-based discount purchases

Key Functions

  • purchaseNode - Allows users to purchase node packages with ERC20 tokens
  • purchaseNodeWithDiscount - Enables discounted purchases using accumulated rewards
  • registerReferrer - Establishes referral relationships and calculates rewards
  • calculateRewards - Determines rewards based on time elapsed and ROI percentage
  • withdrawRewards - Allows users to claim their earned rewards
  • distributeProsperityFund - Distributes the community fund to eligible users

Admin Panel

The admin panel provides an interface to manage the smart contracts on the Base Sepolia testnet. It shows contract information, node packages, referral settings, and prosperity fund details.

Current Default Settings

  • Seven-Level Referral Percentages: 10%, 3%, 2%, 2%, 1%, 1%, 1%
  • Bulk Referral Bonus: 10% after every 10 referrals
  • Prosperity Fund: 10% of purchase price, distributed every 30 days
  • Rewards Discount: Up to 90% discount based on available rewards
