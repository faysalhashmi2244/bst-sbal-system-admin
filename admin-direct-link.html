Admin Panel for Node Packages Contract

The admin panel allows you to manage all aspects of the NodePackages contract deployed on the Base Sepolia testnet.

You can access it through the button below.

Access Admin Panel

Features:

  • Manage seven-level referral percentages
  • Configure prosperity fund settings
  • Set node package active/inactive status
  • Update bulk referral and rewards discount settings
  • Withdraw tokens from the contract
